Consul docker container will be created using the Dockerfile.

Note: For latest or required version of Consul please refer to https://releases.hashicorp.com/consul/

Steps
1.Git clone
2.Ensure require consul version exists in Dockerfile, if not update with valid url
3.Ensure docker is installed
4.Run command docker build -t consul-centos7 . (run command after cd to centos7 directory)
